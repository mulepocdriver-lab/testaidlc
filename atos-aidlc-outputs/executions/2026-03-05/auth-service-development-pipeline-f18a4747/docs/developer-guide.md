# Developer Guide
## User Authentication Service

## Quick Start

### Prerequisites

- Node.js 20.x LTS
- PostgreSQL 15
- Redis 7.x
- Docker (optional)

### Installation

```bash
# Clone the repository
git clone https://github.com/org/user-authentication-service.git
cd user-authentication-service

# Install dependencies
npm install

# Set up environment
cp .env.example .env

# Generate Prisma client
npx prisma generate

# Run database migrations
npx prisma migrate dev

# Start development server
npm run dev
```

### Environment Variables

| Variable | Description | Default |
|----------|-------------|---------|
| DATABASE_URL | PostgreSQL connection string | - |
| REDIS_URL | Redis connection string | redis://localhost:6379 |
| JWT_PRIVATE_KEY_PATH | Path to JWT private key | ./keys/private.key |
| JWT_PUBLIC_KEY_PATH | Path to JWT public key | ./keys/public.key |
| PORT | Server port | 3000 |

### Generate JWT Keys

```bash
# Generate RSA key pair
openssl genrsa -out keys/private.key 2048
openssl rsa -in keys/private.key -pubout -out keys/public.key
```
